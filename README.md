# قوانين وتشريعات اليمن 🇾🇪⚖️

مكتبة قانونية يمنية كاملة على أندرويد — تعمل **بدون إنترنت** بالكامل.

النصوص من المصدر الرسمي: [الجهاز المركزي للرقابة والمحاسبة](https://www.cocayemen.com/laws) (~280 قانوناً وتشريعاً من 1990 حتى اليوم).

> 📖 **للبدء فوراً:** اتبع [دليل التشغيل SETUP.md](SETUP.md)

---

## المميزات — المرحلة الأولى

- 📚 **المكتبة الكاملة** — تصفح كل القوانين حسب السنة أو المجال (19 تصنيفاً)
- 🔍 **بحث شامل دون اتصال** — في نصوص كل المواد مع تظليل النتائج، ويتفهم اختلاف الهمزات والتاء المربوطة
- 📄 **قارئ فاخر للقانون** — مواد بأسلوب الجريدة الرسمية، بحث داخل القانون، تحكم بحجم الخط
- ⭐ **مفضلة** — احفظ المواد المهمة وارجع لها بلمسة
- ⎘ **نسخ المادة** — مع اسم القانون والمصدر جاهزة للمشاركة
- 🛠 **إصلاح آلي للنص العربي** — معالجة انعكاس "لا/ال" الشهير في ملفات PDF الحكومية

## البنية

```
├── www/                  التطبيق (HTML/CSS/JS — بدون مكتبات)
│   └── data/             قاعدة البيانات (يولّدها الـ Scraper)
├── scraper/              سحب وإصلاح وتقسيم القوانين
│   ├── scrape.js         السحب من cocayemen.com
│   └── arabic-fix.js     إصلاح أعطال استخراج PDF العربية
├── .github/workflows/    السحب والبناء على GitHub Actions
├── workflows-backup/     نسخة ظاهرة من الـ workflows (للرفع اليدوي)
└── capacitor.config.json إعدادات تطبيق أندرويد
```

## التقنيات

Capacitor 6 · Vanilla JS · GitHub Actions (سحب + بناء APK) · pdftotext

## خارطة الطريق

| المرحلة | المحتوى | الحالة |
|---------|---------|--------|
| 1 | المكتبة الكاملة + البحث دون اتصال | ✅ هذا الإصدار |
| 2 | المساعد القانوني الذكي (تحليل مشكلات مع استشهاد إلزامي بالمواد) | قادم |
| 3 | الحاسبات القانونية (مواريث، نهاية خدمة، مدد طعن) + منشئ المستندات | قادم |
| 4 | النسخة الموحدة للقوانين المعدّلة + خريطة القضية | قادم |

## تنبيه قانوني

هذا التطبيق أداة اطلاع وبحث ولا يقدم استشارة قانونية ولا يغني عن محامٍ مرخّص. النصوص مستخرجة آلياً وقد تحوي أخطاء — المرجع النهائي هو النص الرسمي في الجريدة الرسمية.
